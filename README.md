# Тестовое задание

# Запуск

Make sure to install the dependencies:
node -v 20.13.1

```bash
# npm
npm install
npm run dev

```
Start the development server on `http://localhost:3000`:

## Production

Build the application for production:

```bash
# npm
npm run build

```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview

# bun
bun run preview
```
